# Compresed-video
Данный проект необходим для сжатия большого количества видео файлов, для уменьшения их объёма. Так же присутствует возможность удалить звук из записи, как частично, так и во всех видео сразу.

  Програма имеет графический интерфейс, понятный большинству пользователей сразу, а так же для работы использует FFmpeg (отдельная установка не требуется, при запуске программы, будет проверка и если не обнаружен файл, он предложит скачать и установить его, или указать путь, куда установлен плагин).
